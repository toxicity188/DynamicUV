# DynamicUV
A simple library for Minecraft Java Edition models with in-game configurable textures. 

### How does it work?
It uses a 1.21.4 modern item model.

### API
```kotlin
repositories {
    maven("https://jitpack.io")
}

dependencies { 
    implementation("com.github.toxicity188:DynamicUV:Tag")
}
```
